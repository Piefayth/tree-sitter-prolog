# tree-sitter-prolog

## How to use

To include in your project.

> npm install --save tree-sitter-prolog

## How to run

* Clone git
* Install dependencies
  > npm install

  > npm install -g node-gyp

* Build
  > npm run build

* Run
  > tree-sitter generate
  > tree-sitter parse examples/example1.pl
  > tree-sitter test
